A high-performance, editorial-style portfolio website built to showcase projects, skills, and certifications with a focus on minimalist aesthetics and premium interactions.

## Key Features

- **Editorial Design:** Large typography and high-density information layout inspired by modern design trends.
- **Interactive Certifications:** 3D flip-card effect for certificates with a responsive auto-scrolling marquee on desktop.
- **Smooth Theme Switching:** Custom Circle Mask View Transition for seamless dark/light mode toggling.
- **WhatsApp Integration:** Direct communication channel integrated into the contact form.
- **Responsive & Performant:** Optimized for all devices using Vite and Tailwind CSS.

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS (Vanilla CSS for custom animations)
- **Animations:** Framer Motion, View Transitions API
- **Icons:** React Icons
- **Deployment:** Vercel / GitHub Pages

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/ArvinSSatria/arvinsultansatria.git
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run development server:**

   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: UI components (Hero, About, Skills, Certifications, etc.)
- `src/context`: State management for Theme/Dark mode.
- `src/assets`: Static assets and images.
- `src/index.css`: Global styles and custom Tailwind utilities.
